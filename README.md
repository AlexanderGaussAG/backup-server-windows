# Usage
<!-- usage -->
```sh-session
$ npm install -g mssqlbackup
$ mssqlbackup COMMAND
running command...
$ mssqlbackup (-v|--version|version)
mssqlbackup/0.0.3 win32-x64 node-v10.16.2
$ mssqlbackup --help [COMMAND]
USAGE
  $ mssqlbackup COMMAND
...
```
